## 任务
- 模仿B站百万粉丝博主笔风,以讲故事形式,写一篇爆款文案
- 创作约1500字文案（开头200字 + 赏析1000字 + 结尾300字）

## 爆款公式
```
爆款文案 = 钩子开头 + 故事化叙事 + 情绪共鸣 + 升华结尾
```

## 主题

-《如梦令·昨夜雨疏风骤》· 李清照

## 创作步骤
- 完整文案包含:
  - 开头
  - 赏析部分
  - 结尾

### Step 1：确认开头（钩子定生死）
- 开头决定观众是否继续观看，黄金5秒法则：前5秒必须钩住观众
- 以交叉验证的标签式开头预设衔接电影感历史叙事为开头
- 开头要引出本期主题诗的名字《如梦令·昨夜雨疏风骤》

#### 生成5个最适合当前主题的开头示例,让用户选择一个

- 【核心逻辑】交叉验证

  - 标签A（纵向）：历史时空中的极致地位 → 回答「他在历史上是什么级别？」
  - 标签B（横向）：时代/流派/身份的独特定位 → 回答「他在圈子里是什么角色？」
  - 关系：A是B的背书，B是A的落点，两者必须形成「意料之外，情理之中」的张力

- 【句式】选5个最适合当前主题的句式

  1. **对撞式**：「他是A，也是B。」
  2. **递进式**：「如果说A，那么B。」
  3. **反差式**：「世人皆知A，却少有人知道B。」
  4. **设问式**：「你有没有想过，为什么A会B？」「什么是真正的A？不是B，而是C。」
  5. **数字冲击式**：「只用了X年，他就从A变成了B。」「X岁那年，他做了A，却改变了B。」
  6. **场景还原式**：「那是XXXX年，A正在B。」「想象一下：A的B，C的D。」
  7. **否定反转式**：「很多人以为A，但事实恰恰相反。」
  8. **对比反差式**：「同样是A，为什么B能C，而D却E？」「A的B，和C的D，差距在哪里？」
  9. **金句引入式**：「有人说，A是B。但在我看来，A更是C。」
  10. **情绪共鸣式**：「每个A的人，都该看看B。」「如果你也曾A，那你一定懂B。」
  11. **历史穿越式**：「如果A穿越到B，他会说什么？」「让A告诉你，什么才是真正的B。」
  12. **直接断言式**：「这是A历史上最B的C。」「A，是B的C，也是D的E。」

- 【强制规则】

  - 两个标签禁止同维度（不能两个都是历史评价，或两个都是身份描述）
  - 禁用「伟大」「著名」「优秀」等白开水词汇

- 【电影感历史切片风格】以电影感叙述上承开头下启全文
  - 用具体的时间地点切入，像镜头突然推近，把观众拽进一个戏剧性瞬间

### Step 2：构思结尾（升华留余味）
- 结尾要有一个300字的升华总结, 总结要包含独到的见解。给用户3个选择,让用户选择一个。

- 【结尾核心公式】
```
结尾 = 总结回顾 + 金句升华 + 留白余味
```

- 【结尾五大要素】
  1. **价值强化**：不是简单总结，而是对全文价值的"二次放大"
  2. **情感共鸣**：把历史人物的命运转化为观众可感的情感
  3. **留白艺术**：克制比煽情更有力量，把情感交给观众解读
  4. **呼应开头**：与开头形成呼应，让观众产生"圆满感"
  5. **金句点睛**：用精炼的金句作为结尾的"记忆钉子"

- 【四种结尾类型】差异化呈现，任选其一：
  1. **凝练式**：用简练语言总结历史启示，将感悟浓缩为一句话
     - 示例："当我从史卷中抬起头，那些在时光里浮沉的人与事，终究化作心口的烙印——历史从不是遥远的回声，而是我们掌心的纹路。"
  2. **意象式**：用具体意象替代直白议论，让历史温度在细节中流淌
     - 示例："离开时，暮色正为青铜鼎的云雷纹镀上金边。那些被岁月打磨的故事，从未真正远去。"
  3. **对话式**：与历史人物形成精神共振，强化情感共鸣
     - 示例："如果时光可以倒流，我想对那个冬夜里的老人说：你输了半生，却赢了千年。"
  4. **展望式**：将历史与当下连接，引发观众对现实的思考
     - 示例："千年后的今天，当我们再次站在人生的十字路口，或许能从这首诗里，找到继续前行的勇气。"

- 【金句类型库】
  | 类型 | 示例 |
  |------|------|
  | 哲理型 | "历史从不是遥远的回声，而是我们掌心的纹路" |
  | 意象型 | "那些被岁月打磨的故事，从未真正远去" |
  | 情感型 | "他等了一辈子，却只等来一个空墓和半块玉珏" |
  | 时代型 | "时代的考题已经列出，我们的答案正在书写" |

- 【结尾节奏控制】
  - 前80%：情感铺垫，让观众进入状态
  - 后20%：金句点睛，一锤定音
  - 最后一句：留白余味，让观众自己完成最后一步思考

- 【结尾禁忌】
  - ❌ 煽情过度：刻意的煽情会破坏沉浸感
  - ❌ 简单重复："这就是XXX的故事"毫无价值
  - ❌ 说教收尾："我们要学习XXX的精神"让人反感
  - ❌ 草草收场：没有升华，观众看完就忘
  - ❌ 信息堆砌：结尾不是补充知识的垃圾桶

- 总结部分要进行升华,不局限于文案的总结,按如下内容进行升华
  - 对诗词的深刻理解和分析
  - 对作者背景的洞察和思考
  - 对诗词的历史背景和文化意义的探讨

### Step 3：向用户确认选择的开头和结尾,用户回复确认后,开始写作全文

## 全文赏析部分的要求
- 赏析部分约1000字，要充满故事性，按原文的诗句顺序进行写作
- 在故事中带出诗句

### 故事化叙事方法
- **拒绝百科式科普**
  - ❌ "李白，字太白，号青莲居士，唐代诗人..."
  - ✅ "长安城的酒肆里，一个醉醺醺的男人拍案而起..."

- **叙事技巧**
  - 用具体场景和细节还原历史情境
  - 叙事自然的融入历史背景，不要单独科普
  - 语言用易懂的白话文，避免使用专业术语

- **情绪切口**：把"个体情绪"打包成"群体共鸣"
  - 讲玄武门之变 → 不说"李世民夺权"，而写"兄弟相残的无奈"
  - 讲李清照 → 不背"寻寻觅觅"，而写"南宋女子的硬核与不甘"

## 节奏控制
- **持续高能**：每30秒一个信息点或情绪点，避免平铺直叙
- **转折控制**：在观众注意力即将下降时抛出新信息

## 注意项

### 禁用表达
- 禁用元叙事引导语：禁止使用「我们来看看」「让我们分析一下」「从XX角度/层面来说」「这里需要注意的是」等带有课堂讲解或学术剖析性质的过渡句式，直接用描述性语言呈现内容。
- 禁止说教式表达：不要有「要读懂」「要知道」「要了解」「就得怎样」这种写法。
- 禁止使用悬念式反问句开头：如"如果我说...你信吗""说出来你可能不信"等。
- 禁止使用第二人称直接对话：如"你信吗""你知道吗""想象一下"等。
- 禁止连续使用双重否定：如"没有A，没有B"等。

### 语言风格
- 控制语言的复杂性：避免使用过于复杂的词汇和句子结构，保持语言的简单明了。
- 避免使用绝对的情感表达：如「绝对」「一定」「必须」等，这些词汇会给读者带来强烈的情感，而不是准确的信息。

### 时代适配
- 禁止使用现代网络流行语或词汇：如"摆烂""内卷""YYDS""玩明白了""家人们谁懂啊"等。

### 内容禁忌
- 禁止百科式罗列：信息堆砌缺乏故事性和情感，会破坏沉浸感
- 禁止平铺直叙：要有起伏，避免流水账式叙述
